## Приоритет 1 — Быстрые победы (1-2 часа каждая)

### 🧮 Калькулятор кормов
**Проблема:** Клиент спрашивает «сколько корма нужно на 100 бройлеров?» — бот отвечает приблизительно.  
**Решение:** Встроить формулу:
- 1 бройлер за 42 дня = 0.35 + 2.0 + 2.5 = **4.85 кг корма**
- 100 голов × 4.85 = **485 кг** 
- 485 / 25 = **20 мешков** (Старт: 2, Рост: 8, Финиш: 10)
- Стоимость: ~18 500₽

**Эффект:** «Вау-фактор» для клиента + автоматический кросс-селл корма при заказе птенцов.

---

### 📦 Товарные наборы (порода → корм → аптечка)
**Проблема:** Клиент заказывает бройлеров, но не знает что ещё нужно.  
**Решение:** Файл `data/product_bundles.json`:
```json
{
  "Кобб-500": {
    "корм": ["Purina Старт (990₽)", "Purina Рост (950₽)", "Purina Финиш (890₽)"],
    "допродажа": ["Аптечка (200₽)"],
    "совет": "На 100 голов нужно ~20 мешков корма и 3-5 аптечек"
  }
}
```
**Эффект:** Увеличение среднего чека на 30-50%.

---

### 📊 Расширение FAQ-cache до 50+ записей
**Проблема:** Вопросы про конкретные даты вывода, сезонность, конкурентов — уходят в LLM.  
**Решение:** Добавить записи:
- «Когда вывод индюков?»
- «Вы лучше/дешевле чем XXX?»
- «Можно ли забрать самовывозом?»
- «Как перевозить птенцов домой?»
- и т.д.

**Эффект:** Ещё 20-30% вопросов отвечаются мгновенно (0.0s вместо 3-5s).

### 🔴 Живые остатки из Bitrix24
**Проблема:** Бот может предложить породу, которой сейчас нет в наличии.  
**Решение:** Перед ответом проверять наличие через Bitrix webhook:
```python
def check_bitrix_stock(product_name):
    resp = requests.get(f"{BITRIX_URL}/crm.product.list.json?filter[NAME]={product_name}")
    quantity = resp.json()["result"][0]["QUANTITY"]
    return quantity > 0
```
**Эффект:** Исключает ситуацию «забронировал, а птицы нет».

---

### 💾 Persistent History (история в БД)
**Проблема:** При перезапуске бота вся история диалогов теряется. Клиент пишет «я вчера спрашивал про мулардов» — бот не помнит.  
**Решение:** Таблица в Neon:
```sql
CREATE TABLE chat_history (
    id SERIAL PRIMARY KEY,
    user_id BIGINT,
    user_name TEXT,
    role TEXT, -- 'user' или 'model'
    content TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);
```
**Эффект:** Бот помнит контекст даже после перезапуска.

## Приоритет 3 — Инновации (4-8 часов)

### 🎙️ Голосовые ответы (Gemini Live TTS)
**Идея:** Отвечать голосовым сообщением, как настоящий менеджер.  
**Аудитория:** Многие фермеры — возрастная аудитория, им удобнее слушать.  
**Технология:** Gemini 2.0 Flash → TTS → aiogram `send_voice()`.  
**Эффект:** Уникальная фича, отличает от любых конкурентов.

---

### 🔍 Гибридный поиск (BM25 + Vector)
**Что:** Добавить лексический поиск параллельно с векторным.  
**Зачем:** Когда клиент ищет по точному названию («Д-107», «Биг-6»), лексический поиск точнее.  
**Как:** Neon поддерживает `tsvector` из коробки — один дополнительный SQL-запрос.  

---

### 🤖 Авто-обучение на реальных диалогах
**Что:** Каждый диалог, где клиент отправил номер телефона (конверсия!), записывается как «идеальный сценарий». LLM анализирует и обновляет SKILL.md.  
**Эффект:** Бот со временем становится лучше, учась на своих же успешных продажах.
