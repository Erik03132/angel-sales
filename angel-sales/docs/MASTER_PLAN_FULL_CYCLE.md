## 🎙️ ГОЛОС АНЖЕЛОЧКИ

### Стратегия «Два Голоса»:

| Режим | Технология | Когда |
|---|---|---|
| ☁️ **Онлайн** | Gemini 3.1 Flash TTS | Есть интернет → премиум голос |
| 🔌 **Оффлайн** | VoxCPM2 (локально) | Нет интернета → локальный голос |

### Провайдер для звонков:
**Zvonok.com** — лучший для нашей задачи:
- 0.10₽/вызов + 2.10₽/мин
- ИИ-робот с распознаванием речи
- Сценарии: «Здравствуйте, ваш заказ на 100 цыплят РОСС-308 подтверждён. Доставка в четверг. Нажмите 1 для подтверждения.»

### Интеграция голоса в воронку:
```
Клиент оставил заявку в чате
    ↓
Анжела создала лид в Битрикс
    ↓
Через 5 минут → Автозвонок (Zvonok.com API):
  «Здравствуйте, [Имя]! Это Азовский Инкубатор.
   Подтверждаю ваш заказ: [порода], [кол-во] штук.
   Доставка [дата]. Для подтверждения нажмите 1.»
    ↓
Если нажал 1 → статус в Битрикс: «Подтверждён»
Если не ответил → SMS через SMSC.ru
```
