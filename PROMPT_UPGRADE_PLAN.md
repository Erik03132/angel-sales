# 🚀 План апгрейда агентов (Prompts.chat Integration)

На основе анализа репозитория [prompts.chat](https://github.com/f/prompts.chat) и текущей архитектуры проекта Freelance Agent 2026, мы внедрим систему «продвинутого промптинга». Это позволит сделать ответы агентов более глубокими, профессиональными и структурированными.

## 1. Внедрение фреймворка C.R.A.F.T.
Использование структуры CRAFT (Context, Role, Action, Format, Target Audience) станет стандартом для всех системных инструкций.

| Элемент | Описание для нашего проекта |
| :--- | :--- |
| **Context** | Описание текущей ситуации на рынке фриланса 2026 года и специфики биржи. |
| **Role** | Четкое определение роли (например, "Ведущий архитектор систем на Node.js с опытом 15 лет"). |
| **Action** | Список конкретных шагов по анализу ТЗ или генерации оффера. |
| **Format** | Жесткое следование `IGOR_STYLE_OFFER.md`. |
| **Target Audience** | Клиент на бирже (зачастую занятой бизнесмен или менеджер проекта). |

## 2. Новая архитектура директории `.agent`
Мы создадим библиотеку «личностей» (Personas), которые можно будет подключать к разным задачам.

```
.agent/
├── prompts/
│   ├── library/            # Библиотека ролей из prompts.chat
│   │   ├── accountant.md    # Для Марины (финансы)
│   │   ├── recruiter.md     # Для анализа вакансий
│   │   ├── prompt_eng.md    # Мета-агент для улучшения промптов
│   │   └── strategist.md    # Бизнес-анализ проектов
│   └── templates/          # Шаблоны CRAFT
└── rules/
    └── GLOBAL_BOOT_PROTOCOL.md
```

## 3. Специфические улучшения агентов

### Агент: Шерлок (Стратег)
*   **Источник:** `prompts.chat/Business Analyst` + `prompts.chat/Debater`
*   **Улучшение:** Глубокая дедукция ТЗ, выявление скрытых болей клиента и поиск нестандартных Open Source решений.

### Агент: Рембрандт (Визуализатор)
*   **Источник:** `prompts.chat/UX/UI Developer`
*   **Улучшение:** Генерация "вау-эффекта" через описание современных дизайн-паттернов (Glassmorphism, Micro-interactions) и соблюдение `CONSTITUTION.md` в защите работ.

### Агент: Кулибин (Инженер/DevOps)
*   **Источник:** `prompts.chat/Linux Terminal` + `prompts.chat/Cyber Security`
*   **Улучшение:** Проектирование неубиваемой инфраструктуры и CI/CD пайплайнов для клиентов.

### Агент: Ботмэн (Интегратор)
*   **Источник:** `prompts.chat/IT Architect`
*   *   **Улучшение:** Сверхсложная логика интеграции AI-агентов и внешних API.

### Агент: Марина (Бухгалтер/Аналитик)
*   **Источник:** `prompts.chat/Accountant`
*   **Улучшение:** Анализ рисков, налоговая оптимизация и расчет ROI проекта.

### Агент: Ангелочка (Супервайзер)
*   **Источник:** `prompts.chat/Motivational Coach`
*   **Улучшение:** Надзор за качеством, мотивация команды и психологическая поддержка Бро.

## 4. Мета-агент "The Architect"
Создадим специальный инструмент (скрипт или отдельный промпт), который будет помогать тебе генерировать новые навыки для агентов, используя логику `Prompt Generator` из prompts.chat.
