# 📋 Передача проекта «Анжелочка» — Инструкция для Андрея

**Проект:** AI-бот «Анжелочка Заботкина» (Telegram + API-сервер)  
**Версия:** v9.0  
**Дата:** Апрель 2026  

---

## 🎯 Что это и зачем

Анжелочка — это AI-помощник в Telegram, который:
- Отвечает клиентам на вопросы о птице, ценах, доставке
- Консультирует по уходу за птенцами (как живой менеджер)
- Собирает номера телефонов для бронирования
- Пересылает все диалоги вам (админу) в реальном времени
- Самообучается на новых вопросах

## ⚠️ ВАЖНО: Что нужно сделать ПЕРЕД передачей

Сейчас бот работает на **моих** аккаунтах и ключах. Чтобы бот стал полностью вашим и не зависел от меня — нужно завести **свои** аккаунты в нескольких сервисах и подставить ключи в файл `.env`.

## 🔑 Аккаунты, которые нужно открыть

### 1. 🤖 Google AI Studio (Gemini API) — ОБЯЗАТЕЛЬНО
**Что это:** Мозг Анжелочки. Через него она думает и отвечает.  
**Стоимость:** Бесплатно (есть лимит ~1500 запросов/день, для вашего объёма хватит с запасом)  
**Как получить:**
1. Зайти на [aistudio.google.com](https://aistudio.google.com)
2. Войти через Google-аккаунт (gmail)
3. Нажать «Get API Key» → «Create API key»
4. Скопировать ключ (начинается с `AIzaSy...`)

**Что вписать в `.env`:**
```
GEMINI_API_KEY=ваш_новый_ключ
```

> ⚠️ Google AI Studio **заблокирован из России**. Для регистрации и получения ключа нужен VPN/прокси с американским IP. После получения ключа — сам ключ работает из России через OpenRouter (см. ниже).

---

### 2. 🌐 OpenRouter — ОБЯЗАТЕЛЬНО
**Что это:** Запасной канал для AI. Если Gemini напрямую не отвечает (из-за блокировки) — бот автоматически переключается на OpenRouter.  
**Стоимость:** Есть бесплатные модели. Платные модели ~$0.10 за 1 млн токенов (~1000 ответов). Реалистичные расходы: **$1-3/месяц**.  
**Как получить:**
1. Зайти на [openrouter.ai](https://openrouter.ai)
2. Зарегистрироваться (можно через Google)
3. Перейти в Keys → Create Key
4. Скопировать ключ (начинается с `sk-or-v1-...`)
5. Пополнить баланс на $5 (хватит на 2-3 месяца)

**Что вписать в `.env`:**
```
OPENROUTER_API_KEY=ваш_новый_ключ
```

---

### 3. 🐘 Neon Database — ОБЯЗАТЕЛЬНО
**Что это:** Облачная база данных. Хранит каталог товаров, FAQ и «память» Анжелочки.  
**Стоимость:** Бесплатно (Free Tier — 0.5 ГБ, для нас хватит на годы)  
**Как получить:**
1. Зайти на [neon.tech](https://neon.tech)
2. Зарегистрироваться (можно через Google/GitHub)
3. Создать проект → Base: PostgreSQL
4. Скопировать Connection String (начинается с `postgresql://...`)

**Что вписать в `.env`:**
```
NEON_DATABASE_URL='ваш_connection_string'
```

> ⚠️ После смены базы данных нужно будет заново загрузить каталог товаров командой:
> ```
> cd agent && python migrate_data.py
> ```

---

### 4. 🛡️ VPS или SOCKS5-прокси — ОБЯЗАТЕЛЬНО (для работы из России)
**Что это:** Telegram Bot API заблокирован из РФ. Бот подключается к Telegram через прокси-сервер.  
**Варианты:**

| Вариант | Стоимость | Надёжность |
|---------|-----------|:----------:|
| **SOCKS5-прокси (текущий)** | ~300₽/мес | ⭐⭐⭐ |
| **VPS в Европе (Hetzner)** | ~350₽/мес (€3.29) | ⭐⭐⭐⭐⭐ |
| **Oracle Cloud Free** | Бесплатно | ⭐⭐⭐⭐ |

**Рекомендация:** Купить самый дешёвый VPS на [Hetzner](https://www.hetzner.com/cloud) и перенести бота туда. Тогда прокси вообще не нужен — бот работает напрямую.

**Если остаётесь на SOCKS5-прокси:**
```
TELEGRAM_PROXY=socks5://логин:пароль@IP:порт
```

**Если переносите на VPS** — удалите строку `TELEGRAM_PROXY` из `.env`, бот будет работать напрямую.

### 5. 📱 Telegram Bot Token — УЖЕ ВАШИЙ
**Что это:** Токен бота в Telegram.  
**Текущий статус:** Уже на вашем аккаунте, менять не нужно.  
**Если нужно пересоздать:** Написать @BotFather → /newbot

```

```

### 6. 📊 Bitrix24 — УЖЕ ВАШ
**Что это:** CRM, откуда берутся цены на товары.  
**Текущий статус:** Вебхук уже от вашего Bitrix. Менять не нужно.

```
BITRIX_WEBHOOK_URL=https://incubird.bitrix24.ru/rest/.../
```

---

### 7. 📢 Avito API — ЖДЁМ ОТ АНДРЕЯ
**Что это:** Доступ к Avito API для управления объявлениями (аудит, оптимизация, создание новых).  
**Текущий статус:** Логин/пароль от аккаунта Авито есть. Ждём от Андрея `client_id` и `client_secret` для API.  

**Как получить API-ключ:**
1. Зайти на [autoload.avito.ru/profile/api](https://autoload.avito.ru/profile/api)
2. Создать приложение (или открыть существующее)
3. Скопировать `client_id` и `client_secret`

**Как передать Анжелочке (САМЫЙ ПРОСТОЙ СПОСОБ):**
> Просто напишите в Битрикс-мессенджер Анжелочке:
> ```
> client_id: ВАШ_ID
> client_secret: ВАШ_СЕКРЕТ
> ```
> Анжелочка сама сохранит ключи и запустит аудит объявлений! 🎯

**Или вручную в `.env`:**
```
AVITO_CLIENT_ID=ваш_id
AVITO_CLIENT_SECRET=ваш_секрет
```

### 8. 📢 VK — УЖЕ ВАШ
**Что это:** Парсер для обучения Анжелочки на обсуждениях ВК.  
**Текущий статус:** Ваш аккаунт. Менять не нужно.

## ❓ Частые вопросы

**Q: Бот перестал отвечать — что делать?**  
A: `bash manage_angela.sh restart` — перезапустить. Если не помогло — `bash manage_angela.sh logs` и прислать лог.

**Q: Как обновить цены в боте?**  
A: Цены тянутся из Bitrix24 автоматически. Если нужно обновить каталог вручную — `python migrate_data.py`.

**Q: Как добавить товар, которого нет в боте?**  
A: Добавить в Bitrix24 → запустить `python migrate_data.py`.

**Q: Бот отвечает неправильно на какой-то вопрос?**  
A: Добавить правильный ответ в файл `data/faq_cache.json` → перезапустить бота.

**Q: Как посмотреть все диалоги?**  
A: Файл `agent/logs/history.md` или дашборд (если запущен): `python dashboard.py` → открыть http://localhost:5055
