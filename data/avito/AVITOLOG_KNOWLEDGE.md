# 🎯 АВИТОЛОГ — База экспертных знаний
## Оптимизация рекламы на Авито для птицеводства (IncuBird)

---

## 1. ПРИНЦИПЫ РАНЖИРОВАНИЯ НА АВИТО

### Факторы, влияющие на позицию объявления:
1. **Актуальность** — свежие объявления выше (перепубликация каждые 3-5 дней)
2. **CTR (кликабельность)** — чем чаще кликают, тем выше поднимается
3. **Полнота заполнения** — все поля, характеристики, фото
4. **Отзывы продавца** — рейтинг магазина влияет на все объявления
5. **Скорость ответа** — Авито повышает быстроотвечающих продавцов
6. **Геолокация** — показ объявлений ближайшим покупателям
7. **Платное продвижение** — x2, x5, x10 поднятие

### Формула эффективности объявления:
```
CVR (конверсия) = Контакты / Просмотры × 100%

Хорошо:  CVR ≥ 2%
Средне:  CVR 0.5% - 2%
Плохо:   CVR < 0.5%
Мёртвое: CVR = 0% (нет просмотров)
```

## 2. ОПТИМИЗАЦИЯ ЗАГОЛОВКОВ

### Структура идеального заголовка:
`[Что продаём] [Порода/Тип] [Ключевое преимущество] [Город/Регион]`

### Примеры для IncuBird:

**❌ Плохо:**
- «Цыплята»
- «Продам бройлеров»
- «Птица на продажу»

**✅ Хорошо:**
- «Цыплята бройлера КОББ-500 | Доставка по Крыму | Гарантия»
- «Суточные индюшата БИГ-6 | Вывод 20 апреля | Азовский инкубатор»
- «Утята мулард 🦆 С фермы | Вакцинированы | Джанкой»

### Правила заголовков:
1. **Первые 5 слов решают** — именно они видны в превью
2. **Порода в заголовке** — фермер ищет конкретную породу
3. **Дата вывода** — для суточной птицы критически важно
4. **Эмодзи** — ⚡🐣🦆🐔 привлекают внимание (1-2 максимум)
5. **Цена в заголовке** — спорно, но «от 60₽/шт» может увеличить CTR

## 3. ОПТИМИЗАЦИЯ ОПИСАНИЙ

### Структура описания:
```
1. ЗАГОЛОВОК-КРЮЧОК (1 строка)
   «Азовский инкубатор — крупнейший производитель Крыма»

2. ЧТО ПРОДАЁМ (2-3 строки)
   Порода, возраст, вес, вакцинация

3. ЦЕНЫ (таблица)
   От 10 шт — 65₽
   От 50 шт — 60₽
   От 100 шт — 55₽

4. ДОСТАВКА (важнейший блок!)
   «Доставка по всему Крыму. Живегруз с кондиционером.»

5. ГАРАНТИИ
   «Вакцинация, ветсправка, гарантия на падёж 3%»

6. ПРИЗЫВ К ДЕЙСТВИЮ
   «Позвоните сейчас — места бронируются!»
```

## 4. ФОТО-ОПТИМИЗАЦИЯ

### Правила для птицеводства:
1. **Первое фото** — крупный план здоровой птицы на чистой подстилке
2. **Второе фото** — масштаб (много птиц, показать объём)
3. **Третье фото** — процесс упаковки/доставки (живогруз)
4. **Четвертое фото** — ветсправки/документы
5. **Инфографика** — цены в картинке (таблица пород + цены)

### Технические требования:
- Минимум 3 фото, оптимум 8-10
- Горизонтальная ориентация 4:3
- Яркое освещение (не тёмные сарайные фото!)
- Без водяных знаков

---

## 5. СТРАТЕГИЯ ЦЕНООБРАЗОВАНИЯ НА АВИТО

### Для IncuBird:
- **Базовая цена** — чуть ниже рынка (привлекает просмотры)
- **Оптовые скидки** — прогрессивная шкала (мотивирует крупный заказ)
- **Сезонная наценка** — весна/лето +10-15%, осень -5%
- **Выделение ценой** — «от 55₽» привлекает больше, чем «65₽»

### Ценовой мониторинг конкурентов:
Регулярно проверять цены на:
- Бройлер КОББ-500 (основной продукт)
- Индюк БИГ-6
- Утка мулард
- Несушки/курочки

---

## 6. СТРАТЕГИЯ ПРОДВИЖЕНИЯ

### Какие объявления продвигать:
1. **Только с CVR > 1%** — иначе деньги на ветер
2. **Сезонные хиты** — весной бройлер, летом индюк
3. **Новые объявления** — первые 3 дня буст + продвижение
4. **НЕ продвигать мёртвые** — сначала оптимизировать, потом продвигать

### Бюджет продвижения:
- **x2** — для поддержания позиции
- **x5** — для конкурентных категорий
- **x10** — только для ТОП-3 самых прибыльных объявлений
- **XL-объявление** — для основных категорий (бройлер, индюк)

---

## 7. РЕГИОНАЛЬНЫЙ ТАРГЕТИНГ (КРЫМ)

### Ключевые города покупателей:
1. Симферополь — крупнейший рынок
2. Севастополь — второй по объёму
3. Джанкой — логистический хаб IncuBird
4. Евпатория, Ялта, Феодосия — курортные, спрос на яйцо/мясо
5. Керчь — самый дальний, дорогая доставка

### Геостратегия:
- Дублировать объявления для разных городов
- Указывать конкретную стоимость доставки до каждого города
- Публиковать в VIP-время (утро 7-9, вечер 18-20)

---

## 8. A/B ТЕСТИРОВАНИЕ

### Что тестировать:
1. **Заголовки** — 3 варианта, лучший через 5 дней
2. **Фото** — менять первое фото раз в неделю
3. **Цены** — «от 55₽» vs «65₽» vs «договорная»
4. **Описание** — короткое vs развёрнутое

### Метод:
- Создать 2-3 одинаковых объявления с разными заголовками
- Через 5 дней сравнить CTR и контакты
- Убрать проигравшие, масштабировать победителя

## 10. KPI ДЛЯ АНДРЕЯ

### Ежемесячные метрики:
- **Общие просмотры** — все объявления суммарно
- **Общие контакты** — звонки + сообщения
- **CVR общий** — контакты / просмотры
- **CAC** — затраты на продвижение / количество сделок
- **ROI рекламы** — (выручка с Авито − затраты) / затраты × 100%

### Целевые показатели:
- CVR > 2% по каждому объявлению
- CAC < 200₽ на 1 контакт
- Мёртвые объявления = 0
- Время ответа на Авито < 5 мин (влияет на ранжирование)
