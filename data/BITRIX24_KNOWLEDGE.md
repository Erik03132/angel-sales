## 📋 ТИПЫ АКТИВНОСТЕЙ (crm.activity.list → TYPE_ID)

> ⚠️ КРИТИЧНО: crm.activity.list возвращает ВСЕ типы активностей!
> Если нужны ТОЛЬКО звонки — фильтруй по TYPE_ID=2 или PROVIDER_ID=VOXIMPLANT_CALL!

| TYPE_ID | Что это | Пример SUBJECT |
|---|---|---|
| **1** | 📧 EMAIL (письмо) | «Письмо от клиента» |
| **2** | 📞 CALL (звонок) | «Исходящий на +7 978 746-30-96» |
| **3** | 🤝 MEETING (встреча) | «Планёрка» |
| **4** | ✅ TASK (задача) | — |
| **5** | 📠 FAX | — |
| **6** | 💬 SMS / Чат | «Чат открытой линии - Telegram» |
| **7** | 📝 NOTE (заметка) | — |

### ⚡ Важно:
- **TYPE_ID=6** — это НЕ только SMS! Сюда попадают **сессии открытых линий** (Telegram, Авито, VK чаты)
- **TYPE_ID=2** — только телефонные звонки через VoxImplant

## 📞 ПРОВАЙДЕРЫ (PROVIDER_ID)

| PROVIDER_ID | Что это | Описание |
|---|---|---|
| **VOXIMPLANT_CALL** | 📞 Телефонные звонки | Реальные звонки через IP-телефонию |
| **CRM_SMS** | 💬 SMS | Отправка SMS через CRM |
| **CRM_WEBFORM** | 📋 Веб-формы | Заявки с сайта |
| **IMOPENLINES_SESSION** | 💭 Открытые линии | Чаты: Telegram, Авито, VK, Онлайн-чат |
| **CRM_EMAIL** | 📧 Email | Письма по email |

## 📊 ПРАВИЛЬНЫЙ ПОДСЧЁТ ДЛЯ ОТЧЁТОВ

### ❌ НЕПРАВИЛЬНО:
```
activities = crm.activity.list()
calls = [a for a in activities if a['TYPE_ID'] == '2']
# Проблема: crm.activity.list отдаёт МАКСИМУМ 50 за раз!
# Если звонков 281 — то это не 281 звонков, а 281 ВСЕХ активностей!
```

### ✅ ПРАВИЛЬНО:
```python
# Для точного подсчёта ЗВОНКОВ: 
voximplant.statistic.get → РЕАЛЬНЫЕ звонки

# Для подсчёта активностей по типам:
crm.activity.list + filter[TYPE_ID]=2 → только звонки
crm.activity.list + filter[PROVIDER_ID]=VOXIMPLANT_CALL → тоже звонки
crm.activity.list + filter[PROVIDER_ID]=IMOPENLINES_SESSION → чаты
```

## 🔑 API-ОСОБЕННОСТИ

### Пагинация:
- Все list-методы отдают **максимум 50 записей** за раз
- Поле `next` — сдвиг для следующей страницы
- Поле `total` — общее количество
- Rate limit: ~2 запроса/сек (pause 0.3-0.5с между запросами)

### Звонки (VoxImplant):
- `voximplant.statistic.get` — статистика звонков (точнее чем crm.activity)
- `CALL_DURATION` — длительность в секундах
- `RECORD_FILE_ID` — ID файла записи на Диске Битрикса
- `disk.file.get` → `DOWNLOAD_URL` — ссылка для скачивания MP3
- Значимые звонки: >30 секунд (остальное — недозвоны, автоответчики)

### Чаты (Открытые линии):
- `PROVIDER_ID = IMOPENLINES_SESSION`
- В `SUBJECT` — имя клиента + канал (Telegram, Авито)
- В `DESCRIPTION` — текст переписки (HTML)
