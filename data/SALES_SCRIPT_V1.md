## ⚠️ ОБНАРУЖЕННЫЕ ПРОБЛЕМЫ

### 1. FAQ Cache = 0% попаданий
Текущий FAQ не покрывает основные вопросы клиентов.
**Решение:** Загрузить эти 10 скриптов в `faq_cache.json`.

### 2. Породы не распознаются (2%)
Анжелочка не понимает «кобб», «росс», «мулларды» в тексте.
**Решение:** Добавить NER-словарь пород в `sales_logic.py`.

### 3. Дубликаты системных сообщений
В трейсах видны сообщения типа `[СИСТЕМНАЯ ИНСТРУКЦИЯ:...]` — это утечка промптов.
**Решение:** Фильтровать системные сообщения в `tg_bot.py`.

## 🎯 ВОРОНКА ПРОДАЖ (State Machine)

```
СОСТОЯНИЕ 1: ПРИВЕТСТВИЕ
  → Клиент пишет → Анжела: «Здравствуйте! Что вас интересует?»
  → Переход: определена порода → СОСТОЯНИЕ 2

СОСТОЯНИЕ 2: ПОДБОР ПОРОДЫ
  → Анжела: рассказывает про породу, спрашивает количество
  → Переход: названо кол-во → СОСТОЯНИЕ 3

СОСТОЯНИЕ 3: РАСЧЁТ ЦЕНЫ
  → Анжела: считает, предлагает доп. товары (корм, аптечка)
  → Переход: клиент согласен → СОСТОЯНИЕ 4

СОСТОЯНИЕ 4: ЛОГИСТИКА
  → Анжела: «Откуда вы? Самовывоз или доставка?»
  → Переход: назван город → СОСТОЯНИЕ 5

СОСТОЯНИЕ 5: КОНТАКТ
  → Анжела: «Оставьте номер, менеджер перезвонит для подтверждения»
  → Переход: получен телефон → СОСТОЯНИЕ 6

СОСТОЯНИЕ 6: БРОНЬ
  → Создаётся лид в Битрикс24
  → Автозвонок через Zvonok.com
  → SMS подтверждение
```
