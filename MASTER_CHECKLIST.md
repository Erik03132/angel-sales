## 🏗️ Архитектура (для будущих проектов)

```
Клиент → Telegram / Bitrix мессенджер
         ↓
  [angelochka_core.get_answer()]
         ↓
  1. FAQ Cache (0.0 сек, бесплатно)
  2. BM25 Search (0.1 сек, бесплатно)
  3. Vector Search Neon (0.3 сек, бесплатно)
  4. Gemini 2.0 Flash (2-5 сек, Free Tier)
  5. OpenRouter Fallback (3-7 сек, ~0.15₽)
         ↓
  Ответ клиенту + Лог в БД + Копия владельцу
```

**Планировщик (20:00 ежедневно):**
1. `bitrix_scanner.py` → скан CRM
2. `daily_report.py` → отчёт Андрею + Игорю в TG
3. `proposal_queue.py` → одно новое предложение Андрею

**Polling-бот (каждые 15 сек):**
- Читает `im.counters.get` (непрочитанные)
- При новом сообщении → `get_answer()` → `im.message.add`
- Ведёт историю диалога per-user
